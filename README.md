# Lista de Tarefas - Projeto de Estudo

Projeto de estudo: lista de tarefas interativa feita com HTML, CSS e JavaScript.

## Funcionalidades
- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- As tarefas ficam salvas mesmo após fechar o navegador
- Interface simples e responsiva

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Objetivo do projeto
- Praticar manipulação do DOM
- Aprender a criar interações simples com JavaScript
- Melhorar habilidades em frontend
